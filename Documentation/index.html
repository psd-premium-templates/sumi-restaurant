Sumi is a clean PSD theme suitable for Restaurant. You can customize it very easy to fit your needs.

1/ Features:

  • 14 Layered PSD
  • 03 Homepage layouts
  • Google Free Fonts
  • Active and Hover stats are included
  • Easily customizable Sketch files
  • Every page is fully layered and organized with proper names
  • Clean and Modern Style
  • Pixel Perfect
  • 1170px Grid System
